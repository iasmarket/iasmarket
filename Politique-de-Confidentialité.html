Politique de Confidentialité

Dernière mise à jour : 24/11/2025

1. Informations sur l’éditeur

Propriétaire de l’application : iasmarket Poster Groupe Bot
Pays : Maroc
Email de contact : aasepto@hotmail.com

2. Introduction

Cette politique de confidentialité décrit comment iasmarket Poster Groupe Bot collecte, utilise et protège les données personnelles obtenues via Facebook Login et l’API Graph afin de permettre la publication dans les groupes Facebook administrés par l’utilisateur.

3. Données collectées via Facebook

L’application accède uniquement aux informations autorisées explicitement par l’utilisateur :

  • public_profile : nom, identifiant, photo de profil
  • groups_access_member_info : liste des groupes que l’utilisateur administre ou rejoint
  • publish_to_groups : autorisation de publier dans les groupes administrés

Nous ne collectons aucune autre donnée Facebook.

4. Données techniques collectées

Pour assurer le fonctionnement de l’application, les données techniques suivantes peuvent être enregistrées :

  • adresse IP
  • type de navigateur
  • date et heure de la requête
  • logs serveur nécessaires au fonctionnement

5. Finalité de l’utilisation des données

Les données collectées sont utilisées uniquement pour :

  • vérifier l’identité de l’utilisateur via Facebook Login
  • afficher les groupes administrés par l’utilisateur
  • permettre la publication dans les groupes via l’API Graph
  • exécuter les fonctionnalités techniques de l’application

Aucune donnée n’est utilisée à des fins publicitaires ou commerciales.

6. Stockage des données

L’application ne conserve aucune donnée personnelle à long terme. Aucun token Facebook n’est stocké sur nos serveurs. Toutes les actions sont exécutées en temps réel.

7. Sécurité

Nous appliquons des mesures techniques pour sécuriser les échanges entre l’utilisateur, Facebook et notre application (HTTPS, pas de stockage permanent, validation des requêtes).

8. Partage des données

Les données ne sont partagées avec :

  • aucun tiers
  • aucun partenaire commercial
  • aucune plateforme externe

9. Droits des utilisateurs

Les utilisateurs peuvent :

  • retirer les permissions sur Facebook
  • révoquer l’accès à l’application via : Paramètres Facebook
  • nous contacter directement pour toute demande de suppression ou d’information

10. Procédure de suppression des données (Facebook Data Deletion)

Conformément aux exigences de Meta, les utilisateurs peuvent demander la suppression de leurs données liées à Facebook.

Pour cela, envoyez une demande à : aasepto@hotmail.com avec pour objet : "Demande de suppression des données Facebook".

La suppression est appliquée sous 48h.

11. Cookies

Le site peut utiliser des cookies essentiels uniquement pour le fonctionnement technique. Aucun cookie publicitaire n’est utilisé.

12. Modifications

Cette politique peut être mise à jour à tout moment. La version la plus récente reste disponible sur cette page.
